### 使用技术：
本博客使用[vuepress-v2](https://v2.vuepress.vuejs.org/zh/)搭建。

### 相关命令：
- 首先`yarn install`下载依赖包（仅第一次时需要）
- `yarn docs:dev` 启动本地环境
- `yarn docs:build` 打包项目，默认打包目录为`./docs/.vuepress/dist`
